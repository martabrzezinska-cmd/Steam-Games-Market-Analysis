# Steam Data Processing Project (2021-2025)

A data science project for processing and analyzing Steam games dataset using Python and Jupyter notebooks.

## 📋 Project Overview

This project provides a comprehensive data processing pipeline for Steam games data, including:
- Data loading and exploration
- Data cleaning and validation
- Data transformation and analysis
- Visualization and insights

## 📁 Project Structure

```
Project/
├── data/                              # Data directory
│   ├── a_steam_data_2021_2025.csv    # Raw Steam data
│   ├── steam_data_invalid_genre.csv   # Test data with invalid genres
│   └── steam_data_processed.csv       # Processed output
├── data_processing.ipynb              # Main processing notebook
├── README.md                          # This file
└── VSCODE_AGENT_GUIDE.md             # Guide for using agents in VS Code
```

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook or JupyterLab
- Required libraries: pandas, numpy, matplotlib, seaborn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/martabrzezinska-cmd/Project.git
   cd Project
   ```

2. Install required dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```

3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook data_processing.ipynb
   ```
